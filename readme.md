Title: FoodApp - Food Delivery Backend

About
A Spring Boot based food delivery backend inspired by Zomato and Swiggy. Built as a learning project covering core backend concepts.
Tech Stack

Java 17
Spring Boot 3.2
Spring Security with JWT
PostgreSQL
Redis
Razorpay Payment Gateway

Features built so far

User authentication with JWT
Restaurant management
Menu item management
Redis based cart
Order placement and status tracking
Razorpay payment integration
